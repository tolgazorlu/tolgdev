## 💻 Proje Kurulum

- Adım 1 - Bu projeyi bilgisayarınıza klonlayın:
  `git clone https://github.com/tolgazorlu/tolgdev`

- Adım 2 - Projede dizinine gidin, gerekli modülleri yükleyin ve projeyi çalıştırın:
  `cd tolgdev`
  `npm install`
  `npm start`
